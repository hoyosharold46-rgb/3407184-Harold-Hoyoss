# 🏛️ Proyecto Semana 1: Ficha de Presentación de Dominio

## 📋 Dominio Asignado

**Dominio:** Plataforma de Participación Ciudadana

Este proyecto presenta una ficha informativa en consola sobre una **propuesta ciudadana**, que es una de las entidades principales dentro de una plataforma digital donde los ciudadanos pueden presentar ideas, votar iniciativas y participar en decisiones públicas.

---

## 🎯 Objetivo del Proyecto

Construir un script en **JavaScript** que se ejecute en **Node.js** y muestre en la consola una ficha de presentación de una entidad del dominio asignado.

El programa demuestra el uso de:

* `console.log()`
* Tipos de datos primitivos (`string`, `number`, `boolean`)
* Comentarios en el código
* Expresiones aritméticas
* Salida organizada en consola

Este proyecto corresponde al **único entregable obligatorio de la Semana 1 del bootcamp**.

---

## 📋 Descripción

El script genera una **ficha digital en consola** que muestra información sobre una propuesta dentro de una plataforma de participación ciudadana.

La ficha incluye:

* Información general de la propuesta
* Datos numéricos relacionados con votos ciudadanos
* Estados booleanos sobre el estado de la propuesta
* Un pequeño resumen del dominio

Todo el contenido se imprime en consola usando únicamente `console.log()`.

---

## 🏛️ Contexto del Dominio

Una **Plataforma de Participación Ciudadana** es un sistema digital que permite a los ciudadanos:

* Presentar propuestas para mejorar su comunidad
* Participar en procesos de votación
* Colaborar en decisiones públicas
* Promover iniciativas sociales o urbanas

La entidad representada en este proyecto es una **propuesta ciudadana**, que contiene información básica como nombre, categoría, número de votos y estado dentro del proceso de revisión.
